# AppNuvem Desktop

Versão desktop do sistema AppNuvem, desenvolvida para permitir manipulação de dados em ambiente local com interface adaptada para computadores.

Este projeto faz parte de um conjunto de aplicações multiplataforma que compartilham a mesma lógica de negócio e banco de dados.

## Objetivo

Fornecer uma alternativa desktop para acesso ao sistema, permitindo:

- gerenciamento de dados
- visualização organizada das informações
- integração com banco de dados remoto
- consistência entre plataformas

## Tecnologias utilizadas

- C#
- .NET
- interface desktop
- integração com banco de dados na nuvem

## Funcionalidades

- cadastro e edição de registros
- exclusão de dados
- sincronização com sistema web e mobile
- reutilização da mesma base lógica do sistema principal

## Contexto acadêmico

Projeto desenvolvido como parte de atividade de criação de sistema multiplataforma durante o curso técnico no SENAI.
